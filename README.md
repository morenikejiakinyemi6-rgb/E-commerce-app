**React Redux Shopping Cart**

This project is a simple e-commerce cart built with React and Redux Toolkit to demonstrate global state management.

**Features**

• Product listing
• Add product to cart
• Increase and decrease product quantity
• Remove items from cart
• Clear entire cart
• Automatic cart total calculation

**Technologies Used**

React
Redux Toolkit
JavaScript (ES6)
CSS

This project was built as part of my learning journey while studying React and Redux state management.
